# MyPersonalShelf

MyPersonalShelf는 내 컴퓨터의 실제 파일, 폴더, 문서, 영상, 음악, 이미지, 메모, 웹 링크를 한곳에 모아두는 로컬 우선 데스크톱 앱입니다. 단순 북마크 앱보다 넓은 개념의 개인 콘텐츠 선반을 목표로 합니다.

영어 문서는 [README.md](README.md)를 참고하세요.

## 왜 만들었나

개인 자료는 보통 다운로드 폴더, 브라우저 북마크, 로컬 폴더, 메신저 첨부파일, 메모 앱, 미디어 플레이어에 흩어져 있습니다. MyPersonalShelf는 이 자료들을 한곳에 복사해서 넣기보다, 내가 원하는 방식으로 다시 묶어 보는 개인용 허브를 실험합니다.

이 앱은 로컬 우선을 기본 방향으로 둡니다.

- 파일을 앱 내부로 복사하기보다 경로와 메타데이터를 먼저 저장합니다.
- Tauri 데스크톱 API로 로컬 파일과 폴더를 엽니다.
- 앱 상태, 읽기 위치, 미디어 재생 위치, 대시보드 배치, 설정을 로컬에 저장합니다.
- 가짜 샘플 폴더를 기본으로 넣지 않고, 빈 선반에서 시작합니다.

## 현재 앱 구조

현재 앱은 다음 페이지들로 구성되어 있습니다.

- **대시보드**: 고정 카드, 최근 열린 항목, 자주 여는 항목, 핸들 드래그 순서 변경, 빈 상태 안내.
- **라이브러리**: 저장한 모든 파일, 링크, 폴더, 노트, 미디어 목록과 미리보기.
- **컬렉션**: 컬렉션 카드, 컬렉션 편집, 태그 한눈에 보기.
- **커스텀**: 색상, 읽기 감각, 대시보드 카드 배치 등 시각적 개인화.
- **설정**: 언어, 열기 방식, 검색 동작, 데이터 내보내기 등 앱 동작 설정.
- **사용 설명서**: 앱의 개념과 기본 사용 흐름 안내.

## 주요 기능

### 콘텐츠 등록

- Tauri 파일 다이얼로그로 로컬 파일 추가.
- Tauri 폴더 다이얼로그로 로컬 폴더 추가.
- 웹 링크 직접 추가.
- 메모와 텍스트 조각 직접 추가.
- 브라우저 세션용 파일 업로드 미리보기.
- 원본 파일을 복사하지 않고 경로와 메타데이터 중심으로 저장.
- 저장한 폴더의 내부 항목 일부 읽기.

### 대시보드

- 즐겨찾기/고정 콘텐츠 카드를 표시합니다.
- 최근 열린 항목과 자주 여는 항목을 보여줍니다.
- 카드 핸들을 끌어 다른 고정 카드 앞에 놓아 순서를 바꿀 수 있습니다.
- 카드 순서, 형태, 밀도, 숨김/표시 상태를 저장합니다.
- 카드 형태는 기본, 가로형, 세로형을 지원하고 카드 밀도는 크게, 보통, 작게 고를 수 있습니다.
- 대시보드 카드 클릭 시 선택한 항목을 라이브러리로 가져오며, 상세 확인과 열기는 라이브러리에서 이어집니다.
- 콘텐츠가 없을 때는 빈 선반 안내를 보여줍니다.
- 상세 선택과 열기는 라이브러리에서 이어지며, 대시보드는 선반 배치에 집중합니다.

### 라이브러리, 컬렉션, 태그

- 저장한 모든 콘텐츠를 한곳에서 봅니다.
- 문서, 영상, 음악, 이미지, 링크, 폴더 타입별 필터를 지원합니다.
- 태그와 컬렉션을 저장하고 수정할 수 있습니다.
- 상세 패널에서 컬렉션과 태그를 바로 편집할 수 있습니다.
- 태그와 컬렉션을 클릭해 라이브러리를 필터링할 수 있습니다.
- 컬렉션 이름, 색상, 아이콘을 편집할 수 있습니다.
- 태그를 한눈에 보는 영역이 있습니다.
- 최근 활동을 우선하는 기본 정렬을 사용합니다.

### 뷰어와 열기 방식

- Tauri로 `txt`, `md`, `markdown`, `log` 등 텍스트 계열 파일을 읽습니다.
- Markdown은 원문 그대로가 아니라 읽기 좋은 블록 형태로 렌더링합니다.
- 문서는 읽기 화면으로 엽니다.
- 영상, 음악, 이미지는 Tauri asset URL을 통해 미리보기/재생합니다.
- 문서 읽기 진행률을 저장합니다.
- 영상/음악 이어보기 위치를 저장합니다.
- 웹 링크는 기본 브라우저로 엽니다.
- 폴더는 시스템 파일 탐색기로 엽니다.
- 설정에 따라 별도 창 또는 앱 안에서 뷰어를 열 수 있습니다.

### 커스텀과 설정

- 배경색, 표면색, 글자색, 강조색을 조정할 수 있습니다.
- 읽기 폭, 줄간격, 글자 크기를 조정할 수 있습니다.
- 고정 카드 밀도를 크게, 보통, 작게 조절할 수 있습니다.
- 대시보드 카드 핸들을 직접 끌어 순서를 바꾸고, 커스텀 화면에서 형태와 숨김 여부를 조정합니다.
- 이전 `compactCards` 설정은 새 카드 밀도 설정으로 자동 변환됩니다.
- 언어는 영어/한국어를 지원합니다.
- 기본 뷰어 열기 방식은 별도 창/앱 안에서 중 선택할 수 있습니다.
- 메인 탭 이동 시 검색 초기화 여부를 설정할 수 있습니다.
- 검색창 Enter 동작을 첫 결과 선택/첫 결과 바로 열기 중 선택할 수 있습니다.
- 현재 앱 데이터를 JSON으로 내보낼 수 있습니다.

### 검색과 빠른 실행

- 제목, 태그, 컬렉션, 경로, 요약을 검색합니다.
- `Ctrl+K`로 검색창에 포커스합니다.
- 명령형 검색을 지원합니다.
  - `tag:reading`
  - `type:video`
  - `collection:Media`
  - `open:novel`
  - `play:focus`
- 설정에 따라 Enter로 첫 결과를 선택하거나 바로 열 수 있습니다.
- `Ctrl+Enter`로 선택된 항목을 엽니다.

## 조작 방식

- **라이브러리 한 번 클릭**: 항목 선택과 미리보기.
- **대시보드 한 번 클릭**: 선택한 항목을 라이브러리로 이동.
- **라이브러리 더블클릭**: 항목 열기.
- **대시보드 카드 핸들 드래그**: 파일/링크 추가 동작과 충돌 없이 고정 카드 순서 변경.
- **Ctrl+Enter**: 선택된 항목 열기.
- **검색창 Enter**: 설정에 따라 첫 결과 선택 또는 열기.
- **마우스 사이드 버튼**: 앱 히스토리 뒤로/앞으로 이동.

## 데이터 저장 방식

- 데스크톱 앱에서는 Tauri를 통해 SQLite에 저장합니다.
- 브라우저 미리보기에서는 localStorage를 fallback으로 사용합니다.
- 로컬 파일은 기본적으로 앱 내부에 복사하지 않습니다.
- 문서 읽기 위치와 미디어 재생 위치는 별도로 저장한 뒤 앱 상태에 반영합니다.
- 대시보드 배치, 컬렉션 설정, 언어, 테마, 앱 설정도 저장됩니다.
- 개발 중 만든 로컬 백업 폴더는 `backups/`로 Git에서 제외합니다.

## 기술 스택

- Desktop: Tauri
- Frontend: React + TypeScript
- Build: Vite
- Local data: SQLite, localStorage fallback
- Styling: 일반 CSS
- Icons: lucide-react
- File dialogs: Tauri + `rfd`
- Database: `rusqlite`

## 개발 방법

### 요구사항

- Node.js
- Rust/Cargo
- Tauri가 지원하는 Windows, macOS, Linux 환경

### 명령어

```bash
npm install
npm run dev
npm run tauri dev
npm run build
npm test
```

`npm run dev`는 Vite 브라우저 미리보기입니다. `npm run tauri dev`는 네이티브 파일/폴더 접근이 가능한 전체 데스크톱 앱입니다. `npm test`는 프론트엔드 회귀 테스트를 실행합니다.

## 프로젝트 메모

- 앱 표시 이름은 `src/lib/appConfig.ts`에 분리되어 있어 나중에 이름을 바꾸기 쉽습니다.
- 초기 선반은 의도적으로 비어 있습니다. 가짜 샘플 콘텐츠 대신 내장 사용 설명서를 제공합니다.
- 현재 구현은 클라우드 동기화나 계정보다 실제 로컬 사용성을 우선합니다.
